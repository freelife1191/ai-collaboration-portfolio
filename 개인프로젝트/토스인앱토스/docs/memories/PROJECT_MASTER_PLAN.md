## 📋 프로젝트 개요

1. **AI 앱**: 토스 앱인토스에서 AI 기반 유틸리티 앱 다수 개발
2. **게임 앱**: 광고에 많이 노출되는 캐주얼 게임 제작
3. **수익화**: 광고(AdMob) + 인앱 결제로 수익 창출

- ✅ **무료 AI API**: Google Gemini, Groq 등 활용
- ✅ **토스 생태계 활용**: 3,000만 누적 사용자 접근

## 🛠️ 제로 비용 기술 스택

### 프론트엔드
- **네이티브 앱**: React Native (Expo)
- **웹 앱**: Next.js 16+ (Node.js 20.9+, TypeScript 5+, React 19.2, Turbopack)
- **게임 엔진**: Phaser.js, PixiJS (React Native 통합)
- **배포**: EAS Build (무료) → 토스 앱인토스 CDN

### 백엔드 (서버리스 100%)
- **우선**: Vercel Edge Functions (100만 실행/월 무료)
- **확장**: Cloudflare Workers (1,000만 요청/월 무료)
- **게임 서버**: Firebase Cloud Functions (200만 호출/월 무료)

### 데이터베이스
- **초기**: Firebase Firestore (1GB + 150만 읽기/월 무료)
- **확장**: Cloudflare D1 (5GB + 1.5억 읽기/월 무료)
- **게임 리더보드**: Firebase Realtime Database (10GB 무료)

### 파일 저장
- **이미지/에셋**: Cloudflare R2 (10GB 무료, 대역폭 무제한)
- **대안**: Firebase Storage (5GB 무료)

### AI/ML (모두 무료)
- **텍스트 생성**: Google Gemini 2.5 (무제한 무료)
- **대화 AI**: Groq Llama 3.1 (14,400 요청/일 무료)
- **음성 인식**: Google Speech-to-Text (60분/월 무료)
- **이미지 생성**: Hugging Face Inference API (무료 Rate Limited)

### 수익화
- **광고**: Google AdMob (react-native-google-mobile-ads)
- **분석**: Google Analytics 4 (완전 무료)
- **결제**: 토스페이먼츠 API, react-native-iap (Google/Apple)

### 비용 발생 시점
| MAU | 월 서버 비용 | 상세 |
|-----|-------------|------|
| **< 10,000** | ₩0 | Firebase + Vercel 무료 티어 |
| **10,000~50,000** | ₩0~10,000 | Cloudflare 마이그레이션 |

## 🎮 게임 앱 개발 계획

### 타겟 게임 분석

#### 광고에 많이 노출되는 게임들
1. **핀 뽑기 (Pin Pull)**: 핀을 뽑아서 보물 획득
2. **주차 퍼즐 (Parking Jam)**: 차량 이동 퍼즐
3. **물 정렬 (Water Sort)**: 같은 색 물 분류
4. **숫자 타워 (Number Tower)**: 숫자 합치기
5. **지렁이 게임**: 길 피하며 먹이 획득
6. **크라우드 시티**: 군중 모으기 러너
7. **화이트아웃 서바이벌**: 생존 전략
8. **로얄매치**: 매치-3 퍼즐

#### 공통 특징
- ✅ **초간단 조작**: 탭, 드래그만으로 플레이
- ✅ **짧은 플레이 타임**: 스테이지당 30초~2분
- ✅ **중독성**: "한 판만 더" 심리
- ✅ **광고 수익화**: 스테이지마다 광고 or 보상형 광고

### Phase 1: 우선 개발 게임 (3개)

#### 1. 핀 뽑기 퍼즐 (Pin Pull Clone)
**개발 기간**: 3주
**장르**: 캐주얼 퍼즐

**게임 플레이**:
- 핀을 순서대로 뽑아 보물 획득
- 용암, 폭탄 등 장애물 피하기
- 물리 엔진 기반 (Matter.js)

**기술 스택**:
```
React Native + Phaser.js
```

- 실수 되돌리기: ₩300 or 광고
- 생명 충전: ₩1,000 or 광고

**수익화**:
- 전면 광고: 스테이지 실패 시
- 보상형 광고: 아이템 획득, 생명 충전
- 인앱 구매:
  - 광고 제거 (₩4,900)
