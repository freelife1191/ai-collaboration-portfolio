## 블로그 기능 요구사항

- 프로필
  - 프로필 이미지를 넣으면 참고 블로그와 동일하게 나오고 하단에 "KWON MINSEUNG" 라고 이름이 표시
  - 이름아래 자기 소개 영역
  - 프로필은 하단은 "© Minseung Kwon All rights reserved." 로 표시

- 블로그는 반응형으로 제작
  - 기본사이즈는 참고하는 블로그 사이트와 동일하게 와이드 사이즈 유지
  - 브라우저와 와이드가 넓어지면 좀 더 넓게 볼 수 있게 사이즈 넓힘

- 한페이지에 보여지는 포스팅 갯수
  - Default 5개
  - 이전 페이지, 이후 페이지 보기는 참고 블로그와 동일하게 구성
  - 이전 페이지 보기 "<- PREV"
  - 이후 페이지 보기 "-> NEXT"

- 블로그 다크 모드, 라이트 모드
  - Default는 시스템 모드
  - 다크모드일땐 달모양 아이콘, 라이트모드일땐 해모양 아이콘 표시
  - 아이콘은 컨텐츠 보기에서도 좌측영역에 보이고 컨텐츠 리스트보기 상태일때도 프로필 우측에 보임
  - 아이콘 클릭시 다른 모드로 전환됨

- 블로그 글 작성 포맷 양식
  - 이미지 임베딩, 동영상 링크 임베딩, 이미지 링크 임베딩, 테이블 표시
  - 그외 다른 부분도 목록 구분 등 참고 블로그랑 동일하게 나오도록 개발

- 컨텐츠 하단
  - Publish 날짜, 태그등 동일하게 나오도록 구성

- Footer 영역
  - 블로그 주인장 소개
  - 우선 "KWON MINSEUNG" 라고 나오도록
