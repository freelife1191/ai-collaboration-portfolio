네, **React와 Next.js는 모든 플랫폼에서 배포 가능**하지만, **Nest.js는 서버리스 플랫폼에서 제한적**입니다. 각 플랫폼의 무료 티어 상황을 정리해드립니다.[1][2][3]

## 플랫폼별 지원 및 무료 티어

### Vercel
**지원:** React ✅, Next.js ✅ (최적화), Nest.js ⚠️ (제한적)[4][2]

**무료 티어:**
- Hobby Plan (개인/비상업 프로젝트)
- 100GB 대역폭
- 무제한 배포
- 6,000 빌드 분
- 100 GB-hours 서버리스 함수[5][4]

Nest.js는 Vercel의 서버리스 함수로 작동시킬 수 있지만, 전체 기능이 제한되며 복잡한 설정이 필요합니다.[2][3][1]

### Netlify
**지원:** React ✅, Next.js ✅, Nest.js ❌ (거의 불가능)

**무료 티어 (2025년 9월 변경):**
- **20 빌드/월** (300 크레딧, 1회 배포당 15 크레딧)
- 100GB 대역폭
- 정적 사이트 중심으로 크게 제한됨[6][7]

Netlify는 정적 사이트와 JAMstack에 최적화되어 있어, Nest.js 같은 백엔드 프레임워크는 지원하지 않습니다.[8][9]

### Cloudflare Pages
**지원:** React ✅, Next.js ✅, Nest.js ❌

**무료 티어:**
- **무제한 정적 요청 및 대역폭**
- 500 빌드/월
- 100 커스텀 도메인
- Workers: 100K 요청/일[10][11][5]

Cloudflare Pages는 정적 사이트와 엣지 함수에 특화되어 있어, Nest.js처럼 복잡한 백엔드 앱은 적합하지 않습니다.[10]

### Leapcell
**지원:** React ✅, Next.js ✅, Nest.js ✅ (Node.js 지원)[12][13]

**무료 티어:**
- **20개 프로젝트 무료**
- Django, Node.js, Go, Rust 지원
- 서버리스 + 전용 머신 모드
- PostgreSQL, Redis 내장[14][12]

Leapcell은 Nest.js를 포함한 Node.js 백엔드를 완전히 지원하는 몇 안 되는 무료 플랫폼입니다.[12]


## 바이브코딩 프로젝트에서 React/Next.js 프론트엔드와 Nest.js 백엔드를 함께 사용시 추천

- **프론트엔드**: Vercel 또는 Cloudflare Pages (무제한 무료)
- **백엔드(Nest.js)**: Leapcell (20개 프로젝트 무료) 또는 Railway, Render 같은 백엔드 전문 플랫폼[8][12]

Vercel/Netlify/Cloudflare는 정적 사이트와 서버리스에 최적화되어 있어 Nest.js 같은 풀스택 백엔드는 적합하지 않습니다.[17][8]


네, **Nest.js 배포에 편하고 무료로 사용 가능한 서비스**가 여러 개 있습니다. 각 서비스의 특징을 정리해드립니다.[1][2]

## Railway

Nest.js를 **가장 쉽게 배포**할 수 있는 플랫폼으로, 공식 가이드를 제공합니다. GitHub 연동 또는 CLI로 몇 분 안에 배포 가능하며, PostgreSQL 데이터베이스도 자동 연동됩니다.[3][4][5]

**무료 티어:**
- Hobby Plan 제공 (월 $5 크레딧)
- GitHub 자동 배포
- PostgreSQL, Redis, MySQL 등 DB 포함
- 환경 변수 자동 참조 기능[4][6]

**배포 방법:**
```bash
railway init
railway add -d postgres
railway up
railway domain
```


## Render

Web Services를 무료로 제공하며, Nest.js 배포가 간단합니다. Railway와 비슷하게 GitHub 연동이 쉽고, 무료 PostgreSQL도 제공합니다.[7][8]

**무료 티어 제공 내용:**
- 월 **750 Free instance hours**
- 15분 미사용 시 스핀다운 (다음 요청 시 자동 재시작, ~1분 소요)
- 월 100GB 아웃바운드 대역폭
- 무료 PostgreSQL (1GB, 30일 제한)[7]

**제한사항:**
- 프로덕션 용도 비추천
- Persistent disk 미지원
- SSH 접근 불가
- 단일 인스턴스만 가능[7]

## Google Cloud Run

**완전 무료 티어**를 제공하며, 서버리스 환경에서 Nest.js를 실행할 수 있습니다. 사용하지 않을 때는 0으로 스케일되어 비용이 발생하지 않습니다.[9][10][11]

**Always Free 티어:**
- **월 2백만 요청** 무료
- 180,000 vCPU-초
- 360,000 GiB-초 메모리
- 북미에서 월 1GB 네트워크 송신[11][9]

**특징:**
- 요청이 없으면 완전 무료 (Cold start 있음)
- Docker 컨테이너 기반 배포
- 자동 스케일링
- 90일 $300 크레딧 제공 (신규 가입 시)[10]



## 바이브코딩 프로젝트에서 Nest.js 백엔드를 무료로 배포할때 추천

1. **Railway**: 가장 간편하고 공식 지원이 우수하며, DB 연동이 자동화되어 있어 개발자 경험이 최고입니다[6][3][4]
2. **Google Cloud Run**: 트래픽이 적은 초기 서비스라면 완전 무료로 사용 가능하며, 확장성이 뛰어납니다[9][11]
3. **Render**: 간단한 테스트나 포트폴리오 프로젝트에 적합하지만, 15분 idle 후 스핀다운이 단점입니다[7]

Railway가 Nest.js 개발자에게 가장 인기 있는 선택이며, GitHub Push 시 자동 배포되는 CI/CD가 내장되어 있어 편리합니다.[3][6][4]

무료 플랜 순위는 다음과 같습니다:

## 1위: Cloudflare Pages

**무료 제공:**
- **무제한 대역폭** (가장 큰 장점)
- 500 빌드/월
- 100개 커스텀 도메인
- 무제한 정적 요청
- Workers: 100K 요청/일[1][2]

**압도적 1위 이유:**
Cloudflare는 자체 CDN 인프라를 사용하므로 트래픽 비용이 거의 없어, **무제한 대역폭**을 제공합니다. 정적 사이트와 SSG에 최적화되어 있으며, React와 Next.js 배포에 완벽합니다.[3][4][1]

## 2위: Vercel

**무료 제공:**
- 100GB 대역폭
- **무제한 배포**
- 6,000 빌드 분
- 100GB-hours 서버리스 함수
- 100K 함수 요청/일[5][1]

**장점:**
Next.js를 만든 회사답게 **최고의 Next.js 최적화**와 개발 경험을 제공합니다. Git 푸시만으로 자동 배포되며, Preview 배포가 강력합니다.[6][4]

## 3위: Google Cloud Run

**무료 제공:**
- 월 **2백만 요청** 무료
- 180,000 vCPU-초
- 360,000 GiB-초 메모리
- 1GB 아웃바운드 전송 (북미)[7][8]

**장점:**
요청이 없으면 **완전 무료**이며, Nest.js 같은 백엔드 앱을 실행할 수 있습니다. 컨테이너 기반으로 모든 언어를 지원합니다.[9][7]

## 4위: Railway

**무료 제공:**
- 월 $5 크레딧 (약 **500시간 기본 인스턴스**)
- 무제한 프로젝트
- PostgreSQL, Redis 등 DB 포함
- Sleep 없음 (항상 실행)[10][11]

**장점:**
Nest.js 배포가 **가장 쉬우며**, GitHub 자동 배포와 DB 자동 연동이 뛰어납니다. 무료 크레딧으로 항상 깨어있는 서버를 운영할 수 있습니다.[11][12][13]

## 5위: Netlify

**무료 제공 (2025년 9월 크게 축소):**
- 월 **20 빌드** (300 크레딧, 배포당 15 크레딧)
- 100GB 대역폭
- 125K 서버리스 함수 요청/월[14][1]

**단점:**
2025년 9월 가격 정책 변경으로 무료 빌드가 300분에서 20회로 **대폭 감소**했습니다. 더 이상 무료로 적극 추천하기 어렵습니다.[15][14]

## 6위: Render

**무료 제공:**
- 750 인스턴스 시간/월
- **15분 idle 후 스핀다운** (재시작 1분 소요)
- 100GB 대역폭
- 무료 PostgreSQL (1GB, 30일 제한)[16][11]

**단점:**
스핀다운으로 인한 콜드 스타트가 가장 큰 단점입니다. 프로덕션 용도는 비추천됩니다.[11][16]

## 7위: Leapcell

**무료 제공:**
- 20개 프로젝트
- Node.js, Python 지원
- PostgreSQL, Redis 내장[17]

**평가:**
신생 서비스로 커뮤니티와 문서가 부족하지만, 20개 프로젝트 무료는 매력적입니다.[17]

바이브코딩 프로젝트에서:
- **React/Next.js 프론트엔드**: Cloudflare Pages (무제한 무료) 또는 Vercel (Next.js 최적화)
- **Nest.js 백엔드**: Railway (쉬운 배포) 또는 Google Cloud Run (완전 무료)[13][7]
