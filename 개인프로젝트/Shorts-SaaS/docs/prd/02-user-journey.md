# User Journey & Personas

## 1. Detailed User Personas

### Persona 1: "성장형 크리에이터 민수" (Growth-Focused Creator)

#### Background
- **이름**: 김민수
- **나이**: 28세
- **직업**: 프리랜서 영상 크리에이터
- **거주지**: 서울
- **수입**: 월 300만원 (YouTube 수익 + 프리랜서)
- **교육**: 대학 졸업 (미디어 전공)

#### YouTube Channel Profile
- **채널명**: "민수의 테크 리뷰"
- **카테고리**: 기술 제품 리뷰, 언박싱
- **구독자**: 15,000명
- **월평균 조회수**: 200,000
- **업로드 빈도**: 주 2회 (수요일, 토요일)
- **평균 영상 길이**: 12-15분
- **수익화 상태**: 활성화 (월 $400-600)

#### Goals & Motivations
1. **단기 목표 (3개월)**:
   - 구독자 25,000명 달성
   - 영상당 평균 조회수 20,000 달성
   - 스폰서십 계약 1건 확보

2. **중기 목표 (1년)**:
   - 구독자 100,000명 달성 (Silver Play Button)
   - 월 수익 $2,000 달성
   - YouTube를 주 수입원으로 전환

3. **장기 목표 (3년)**:
   - 구독자 500,000명
   - 개인 브랜드 확립
   - 다중 수익 채널 구축 (광고, 제휴, 굿즈)

#### Pain Points
1. **시간 부족**:
   - 스크립트 작성에 영상당 4-5시간 소요
   - 편집까지 하면 하루 종일 소요
   - 다른 프리랜서 일과 병행 어려움

2. **콘텐츠 기획 어려움**:
   - 어떤 제품을 리뷰할지 선택 고민
   - 경쟁 채널이 많아 차별화 어려움
   - 트렌드 파악 시간 부족

3. **데이터 분석 부담**:
   - YouTube Studio 데이터가 복잡함
   - 어떤 지표를 봐야 할지 모름
   - 경쟁 채널 분석 시간 없음

4. **성장 정체**:
   - 최근 3개월간 구독자 증가율 감소
   - 조회수가 예측 불가능
   - 어떻게 개선해야 할지 막막함

#### Technology Usage
- **일일 사용 앱**: YouTube Studio, Premiere Pro, Notion
- **스마트폰**: iPhone 14 Pro
- **컴퓨터**: MacBook Pro M2
- **기술 숙련도**: 높음 (신기술 빠르게 습득)

#### Behavioral Patterns
- **작업 시간**: 주로 저녁 8시-새벽 2시
- **콘텐츠 소비**: YouTube, 네이버 블로그, Reddit
- **구매 결정**: 기능 > 가격 (생산성 향상이면 지불 의향 높음)
- **커뮤니티**: 크리에이터 커뮤니티 활발히 참여

#### Quotes
> "스크립트 작성이 가장 힘들어요. 매번 빈 화면을 보고 시작하는 게 너무 부담스러워요."

> "내 채널이 왜 성장이 정체됐는지 알고 싶어요. 데이터는 많은데 뭘 봐야 할지 모르겠어요."

> "경쟁 채널들은 다 잘되는데, 나만 안 되는 것 같아요. 뭐가 다른 걸까요?"

### Persona 2: "신규 크리에이터 지혜" (Newbie Creator)

#### Background
- **이름**: 박지혜
- **나이**: 24세
- **직업**: 대학생 (4학년)
- **거주지**: 부산
- **수입**: 알바 + 부모님 지원
- **교육**: 대학 재학 (경영학 전공)

#### YouTube Channel Profile
- **채널명**: "지혜의 브이로그"
- **카테고리**: 일상 브이로그, 대학생활
- **구독자**: 2,300명
- **월평균 조회수**: 15,000
- **업로드 빈도**: 주 1회 (불규칙)
- **평균 영상 길이**: 8-10분
- **수익화 상태**: 미달성 (구독자 1,000명 달성했으나 시청시간 부족)

#### Goals & Motivations
1. **단기 목표 (3개월)**:
   - 구독자 5,000명 달성
   - 시청시간 4,000시간 달성하여 수익화
   - 안정적인 업로드 루틴 만들기

2. **중기 목표 (1년)**:
   - 구독자 10,000명
   - 월 수익 $300-500
   - 졸업 후 YouTube 창업 가능성 타진

3. **장기 목표 (3년)**:
   - 풀타임 크리에이터로 전환
   - 개인 브랜드 확립
   - 강연 및 컨설팅 사업 확장

#### Pain Points
1. **채널 성장 전략 부재**:
   - 무엇을 해야 구독자가 늘어날지 모름
   - 썸네일, 제목 어떻게 만들어야 할지 막막함
   - 알고리즘이 어떻게 작동하는지 이해 안 됨

2. **콘텐츠 아이디어 고갈**:
   - 매주 뭘 찍을지 고민
   - 일상 브이로그가 반복적이라 지루함
   - 새로운 포맷 시도하고 싶지만 용기 부족

3. **기술적 어려움**:
   - 편집 스킬 부족 (기본 컷 편집만 가능)
   - 스크립트 없이 즉흥으로 촬영해서 산만함
   - 조명, 음향 등 기술적 지식 부족

4. **시간 관리**:
   - 학업과 병행 어려움
   - 촬영-편집-업로드 일정 관리 힘듦
   - 꾸준함 유지가 가장 큰 도전

#### Technology Usage
- **일일 사용 앱**: Instagram, TikTok, YouTube, Notion
- **스마트폰**: Galaxy S23
- **컴퓨터**: LG 그램 (편집용)
- **기술 숙련도**: 중간 (필요한 것만 학습)

#### Behavioral Patterns
- **작업 시간**: 주로 주말, 낮 시간
- **콘텐츠 소비**: YouTube, Instagram Reels, TikTok
- **구매 결정**: 가격 민감 (무료 또는 저렴한 옵션 선호)
- **커뮤니티**: 온라인 크리에이터 커뮤니티 lurker

#### Quotes
> "채널을 시작했는데, 이제 뭘 해야 할지 모르겠어요. 영상은 올리는데 조회수가 안 나와요."

> "매주 뭘 찍을지 너무 고민돼요. 다른 크리에이터들은 아이디어가 어디서 나오는 걸까요?"

> "스크립트를 써야 한다는 건 알겠는데, 어떻게 쓰는지 몰라요. 템플릿 같은 게 있으면 좋겠어요."

### Persona 3: "기업 마케터 상훈" (Corporate Marketer)

#### Background
- **이름**: 이상훈
- **나이**: 35세
- **직업**: 마케팅 매니저 (중견 IT 기업)
- **거주지**: 판교
- **수입**: 연봉 7,000만원
- **교육**: 대학원 졸업 (마케팅 전공)

#### YouTube Channel Management
- **관리 채널 수**: 3개
  1. 메인 브랜드 채널 (구독자 50K)
  2. 제품 튜토리얼 채널 (구독자 20K)
  3. 채용 브랜딩 채널 (구독자 5K)
- **팀 구성**: 마케터 2명, 영상 제작자 1명, 외주 작가
- **월 예산**: 영상 제작 500만원, 광고 1,000만원
- **KPI**: 브랜드 인지도, 리드 생성, 채용 지원자 증가

#### Goals & Motivations
1. **비즈니스 목표**:
   - YouTube를 통한 리드 생성 월 100건
   - 브랜드 인지도 향상 (브랜드 검색량 +30%)
   - 제품 데모 영상 시청 후 전환율 5%

2. **운영 목표**:
   - 채널 관리 효율화
   - 콘텐츠 ROI 측정 체계화
   - 팀 협업 프로세스 개선

3. **경력 목표**:
   - YouTube 마케팅 전문성 입증
   - 회사 내 디지털 마케팅 리더십 확보
   - 업계 컨퍼런스 발표 기회

#### Pain Points
1. **여러 채널 관리의 복잡성**:
   - 3개 채널의 데이터를 각각 확인해야 함
   - 채널별 성과 비교 어려움
   - 통합 리포트 만들기 번거로움

2. **ROI 측정 어려움**:
   - YouTube 성과를 비즈니스 지표로 변환 어려움
   - 경영진에게 보고할 수치 정리 힘듦
   - 예산 대비 효과 입증 필요

3. **팀 협업 비효율**:
   - 작가, 편집자, 마케터 간 소통 단절
   - 스크립트 피드백 주고받기 번거로움
   - 버전 관리 혼란 (이메일, 메신저 산재)

4. **콘텐츠 기획 부담**:
   - 월 12개 영상 기획해야 함 (채널당 4개)
   - 트렌드 파악하면서 브랜드 메시지도 전달해야 함
   - 외주 작가 브리핑 시간 소요

#### Technology Usage
- **일일 사용 툴**: Google Analytics, YouTube Studio, Slack, Notion, Figma
- **마케팅 툴**: HubSpot, Google Ads, Meta Business Suite
- **기술 숙련도**: 높음 (다양한 SaaS 툴 능숙)

#### Behavioral Patterns
- **작업 시간**: 평일 9시-6시 (야근 가끔)
- **의사결정**: 데이터 기반, 팀 협의
- **구매 결정**: 비용 대비 효과, 팀 생산성 향상
- **예산 승인**: 분기별 검토, ROI 입증 필요

#### Quotes
> "3개 채널을 돌아다니면서 데이터 확인하는 게 너무 비효율적이에요. 한 곳에서 보고 싶어요."

> "경영진한테 'YouTube 잘하고 있다'를 어떻게 보여줄지 고민이에요. 조회수만으로는 부족해요."

> "외주 작가분과 협업할 때마다 메신저로 스크립트 주고받는데, 버전 관리가 엉망이에요."

## 2. User Journey Maps

### Journey 1: 민수의 첫 스크립트 생성 (First Script Generation)

#### Scenario
민수는 새로운 제품 리뷰 영상을 촬영하기 전에 스크립트가 필요합니다. 평소에는 4-5시간 걸리는 작업이지만, AI 도구를 사용해보기로 했습니다.

#### Journey Stages

**1. Awareness (인지)**
- **Touchpoint**: YouTube 크리에이터 커뮤니티 글
- **Action**: "AI로 스크립트 자동 생성" 광고 클릭
- **Thoughts**: "이게 정말 될까? 한번 써볼까?"
- **Emotions**: 😐 회의적, 호기심
- **Pain Points**: 스크립트 작성 시간 부족, 마감 압박

**2. Consideration (고려)**
- **Touchpoint**: 랜딩 페이지
- **Action**: 기능 설명 읽기, 예시 스크립트 확인
- **Thoughts**: "무료로 5번 써볼 수 있네. 일단 시도해볼만해."
- **Emotions**: 😊 긍정적, 기대
- **Pain Points**: 유료 결제 부담 (하지만 무료 체험 있음)

**3. Onboarding (온보딩)**
- **Touchpoint**: 회원가입 페이지
- **Action**: Google 계정으로 로그인
- **Thoughts**: "간단하네. YouTube 연동도 쉽고."
- **Emotions**: 😊 만족
- **Pain Points**: 없음 (원클릭 로그인)

**4. First Use (첫 사용)**
- **Touchpoint**: 스크립트 생성 페이지
- **Action**:
  1. 주제 입력: "갤럭시 S24 언박싱 및 첫인상 리뷰"
  2. 영상 유형 선택: 롱폼
  3. 시청 시간: 12분
  4. 톤앤매너: 전문적
  5. "생성" 버튼 클릭
- **Thoughts**: "입력 항목이 직관적이네. 어렵지 않아."
- **Emotions**: 😃 기대, 약간의 긴장
- **Pain Points**: AI가 내 스타일을 이해할까?

**5. Aha Moment (깨달음)**
- **Touchpoint**: 생성된 스크립트
- **Action**: 실시간 스트리밍으로 스크립트 확인
- **Thoughts**: "와, 이거 내가 쓴 것처럼 자연스러운데? 구조도 탄탄하네!"
- **Emotions**: 😍 놀라움, 감탄
- **Pain Points**: 없음 (기대 초과)
- **Key Metric**: Time to Aha Moment = 30초 (생성 완료 시점)

**6. Continued Use (지속 사용)**
- **Touchpoint**: 스크립트 편집기
- **Action**:
  1. 일부 문구 수정 (개인 스타일 반영)
  2. 저장
  3. 촬영 때 사용
- **Thoughts**: "편집도 쉽네. 이제 스크립트는 10분이면 끝나겠어."
- **Emotions**: 😊 만족, 안도
- **Pain Points**: 없음

**7. Habit Formation (습관화)**
- **Touchpoint**: 대시보드
- **Action**: 다음 영상도 AI로 스크립트 생성
- **Thoughts**: "이제 이거 없이는 못 하겠는데? 유료 결제할까?"
- **Emotions**: 😊 신뢰, 의존
- **Pain Points**: 무료 5회 한도 (유료 전환 계기)

**8. Conversion (전환)**
- **Touchpoint**: 가격 페이지
- **Action**: Pro 플랜 결제 ($19.99/month)
- **Thoughts**: "월 2만원으로 시간 절약하고 품질도 올리면 충분히 가치 있어."
- **Emotions**: 😊 확신, 만족
- **Pain Points**: 없음

**9. Advocacy (옹호)**
- **Touchpoint**: 크리에이터 커뮤니티
- **Action**: 플랫폼 후기 작성, 친구에게 추천
- **Thoughts**: "주변 크리에이터들한테 알려줘야겠어."
- **Emotions**: 😍 열성적
- **Pain Points**: 없음

#### Journey Metrics
- **Time to First Value**: 5분 (회원가입부터 첫 스크립트 생성까지)
- **Time to Aha Moment**: 5분 30초
- **Free to Paid Conversion Time**: 3일 (5회 사용 후)
- **Satisfaction Score**: 9/10

### Journey 2: 지혜의 채널 성장 여정 (Channel Growth Journey)

#### Scenario
지혜는 채널이 정체기를 겪고 있습니다. 구독자가 2,000명대에서 멈춰있고, 무엇을 해야 할지 모릅니다.

#### Journey Stages

**1. Problem Recognition (문제 인식)**
- **Touchpoint**: YouTube Studio (구독자 그래프)
- **Action**: 최근 3개월 구독자 증가 거의 없음 확인
- **Thoughts**: "뭔가 잘못되고 있어. 도움이 필요해."
- **Emotions**: 😞 좌절, 불안
- **Pain Points**: 해결 방법 모름, 전문가 조언 필요

**2. Solution Search (해결책 검색)**
- **Touchpoint**: Google 검색 "유튜브 채널 성장 도구"
- **Action**: 여러 도구 비교, 리뷰 읽기
- **Thoughts**: "너무 많아서 뭘 선택해야 할지 모르겠어."
- **Emotions**: 😕 혼란, 압도
- **Pain Points**: 정보 과부하

**3. Platform Discovery (플랫폼 발견)**
- **Touchpoint**: YouTube 광고
- **Action**: 광고 클릭, 랜딩 페이지 방문
- **Thoughts**: "한국어로 돼있고, 무료 체험도 있네. 일단 해볼까?"
- **Emotions**: 😊 희망, 기대
- **Pain Points**: 여전히 반신반의

**4. Onboarding (온보딩)**
- **Touchpoint**: 가입 및 튜토리얼
- **Action**: Google 로그인, 온보딩 튜토리얼 시청
- **Thoughts**: "생각보다 간단하네. 따라할 수 있겠어."
- **Emotions**: 😊 안도
- **Pain Points**: 튜토리얼이 너무 길면 지루할 수 있음

**5. Data Analysis (데이터 분석)**
- **Touchpoint**: 대시보드
- **Action**: 채널 분석 데이터 확인

#### Scenario
상훈의 팀은 3개 채널에 월 12개 영상을 제작합니다. 외주 작가, 편집자, 마케터 간 협업이 필요합니다.

#### Journey Stages

**1. Pain Point Identification (문제점 파악)**
- **Touchpoint**: 팀 회의
- **Action**: 현재 워크플로우 문제점 논의
- **Thoughts**: "스크립트 피드백 주고받는 게 너무 비효율적이야."
- **Emotions**: 😤 답답함
- **Pain Points**: 이메일/슬랙에 스크립트 산재, 버전 관리 혼란

**2. Tool Evaluation (도구 평가)**
- **Touchpoint**: SaaS 리뷰 사이트
- **Action**: 여러 YouTube 도구 비교
- **Thoughts**: "팀 협업 기능이 있는 도구가 필요해."
- **Emotions**: 😐 분석적
- **Pain Points**: 대부분 개인 크리에이터용, 팀 기능 부족

**3. Demo Request (데모 요청)**
- **Touchpoint**: 영업 팀 미팅
- **Action**: Business 플랜 데모 요청
- **Thoughts**: "우리 팀 워크플로우에 맞을까?"
- **Emotions**: 😐 신중함
- **Pain Points**: 의사결정에 여러 이해관계자 필요

**4. Trial Period (체험 기간)**
- **Touchpoint**: 14일 무료 체험
- **Action**: 팀원 초대, 실제 프로젝트 진행
- **Thoughts**: "한번 써보면서 판단하자."
- **Emotions**: 😊 기대
- **Pain Points**: 팀원 교육 필요

**5. Workflow Integration (워크플로우 통합)**
- **Touchpoint**: 팀 워크스페이스
- **Action**:
  1. 외주 작가가 AI로 초안 생성

- **Emotions**: 😊 만족

## 5. Jobs-to-be-Done (JTBD)

### Primary Jobs

**1. "빠르게 질 좋은 스크립트를 만들고 싶다"**
- **Functional**: 영상 제작 시간 단축

**2. "내 채널이 왜 성장하지 않는지 알고 싶다"**
- **Functional**: 데이터 기반 의사결정
- **Emotional**: 불안 해소, 통제감 회복
- **Social**: 성공한 크리에이터로 보이기

**3. "경쟁 채널과 비교해서 내가 어디 위치인지 알고 싶다"**
- **Functional**: 벤치마킹, 전략 수립
- **Emotional**: 뒤처지지 않는다는 안도감
- **Social**: 업계 트렌드 따라잡기

**4. "팀원들과 효율적으로 협업하고 싶다"**
- **Functional**: 워크플로우 개선
- **Emotional**: 스트레스 감소
- **Social**: 좋은 리더로 보이기

### Jobs-to-be-Done Framework

#### Job Statement Template
"When I [situation], I want to [motivation], so I can [expected outcome]."

**Examples:**

1. **Script Generation Job**
   - When I have a video idea but no script,
   - I want to quickly generate a professional script,
   - So I can start filming without writer's block.

2. **Analytics Job**
   - When I upload a new video,
   - I want to understand which parts are performing well,
   - So I can optimize future content.

3. **Growth Job**
   - When my channel growth stagnates,
   - I want to identify what's holding me back,
   - So I can make data-driven improvements.
