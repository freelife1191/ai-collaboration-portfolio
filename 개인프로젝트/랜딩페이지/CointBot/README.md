# 코인봇 스튜디오 랜딩 페이지 제작

- AI툴
  - ChatGPT: 기획 및 화면 설계서 작성
  - v0: 웹사이트의 디자인 및 기본 틀 잡기 (웹사이트 초안)
  - Cursor: 디자인 세부사항 수정 및 기능 추가 (웹사이트 고도화)
  - Notion API: 노션과 연동하여 데이터베이스 활용
  - Vercel: 프로젝트 배포 및 추적
  - Make: 메일 자동화
- 배포: vercel
- 소요시간: 4시간
- GitHub: https://github.com/freelife1191/coinbot-landing-vive
- Demo: https://coinbot-landing.vercel.app/


**요약**

- Cursor AI와 V0를 활용해 바이브코딩으로 랜딩페이지 만들기 실습

**성과**

- 적절한 프롬프트를 입력해 GPT로 기획서와 화면설계서 작성
- v0를 통한 UI 개발
- 레퍼런스 UI 이미지를 활용해 UI/UX 퀄리티를 높임
- 폰트 적용 및 SEO 최적화
- Vercel 배포, Analytics 및 Speed Insights 연동
- Gmail API, Notion API 연동 및 Make를 활용한 이메일 자동화 적용
- 퀄리티 높은 랜딩페이지 제작 노하우를 습득
