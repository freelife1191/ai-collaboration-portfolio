## 화면 설계서

```markdown
---

# 📐 화면 설계서 (페이크 도어 테스트 랜딩 페이지)

## 1. 페이지 구조 (원페이지 스크롤형)

1. **Hero 섹션**

   * 상단 로고 + 네비게이션 미니 메뉴(How it works / Features / Pricing / FAQ)
   * 메인 카피:

     * “전략은 AI에게, 실행은 자동으로.”
     * 서브 카피: “코인봇 스튜디오 — 나만의 AI 코인 매매봇을 만들고, 구독하고, 수익을 공유하는 올인원 플랫폼”
   * CTA 버튼(강조): **\[🚀 웨이팅 리스트 등록하기]**
   * 거래소 로고(Upbit, Bithumb, Bitget) 아이콘 배치 → 신뢰 확보
   * 하단 리스크 고지 문구(작게):
     “가상자산 투자는 원금 손실 가능. 과거 성과는 미래를 보장하지 않음.”

---

2. **How it works (3단계 소개)**

   * Step 1: **전략 생성** (대화형 가이드) → 일러스트
   * Step 2: **백테스트 확인** (성과 곡선 미리보기) → 그래프 모형
   * Step 3: **배포 & 구독** (원클릭 실행) → 카드형 UI
   * 사용 시나리오 예시 카드:

     * “김트레이더는 단 5분 만에 AI 봇을 생성해 첫 달 12% 수익을 경험했습니다.”

---

3. **실시간 차트/데모 영역 (Fake UI)**

   * 인터랙티브한 느낌의 **캔들 차트 뷰** (실제 동작 X, 이미지/모형)
   * 지표 토글 버튼 (RSI, MACD, 이동평균선) → 클릭 시 더미 툴팁 표시
   * 김치 프리미엄 위젯(보라색 포인트)

---

4. **랭킹 & 검증 섹션**

   * 상위 봇 카드(더미 데이터):

     * 수익률 +25%, MDD -8%, 구독자 1,240명
     * 검증 배지(✅ 백테스트, 🔒 실계좌, 🏆 상위랭킹)
   * “투명한 성과 지표로 안심하세요” 카피

---

5. **마켓 & 보안/연동**

   * 템플릿 카드 4개 그리드 (태그: 모멘텀/스윙/퀀트, 프리미엄 라벨)
   * 거래소 연동 아이콘(Upbit, Bithumb, Bitget) + “원클릭 API 연동” 카피
   * 보안 요소 아이콘: 암호화, 키 분리, 권한 설정

---

6. **가격 정책 섹션**

   * Free / Pro / Premium 3단 테이블

     * Pro 플랜 카드 강조 (하이라이트 컬러)
   * CTA 버튼: **\[지금 등록하고 출시 알림 받기]**
   * 리스크 고지 문구 재노출

---

7. **후기/FAQ**

   * 더미 사용자 리뷰 카드(별점 + 짧은 카피)
   * FAQ 아코디언 4개 (수수료 / 환불 / 보안 / 전략 업데이트)

---

8. **웨이팅 리스트 등록 흐름**

   * CTA 버튼 클릭 → 모달 폼 노출

     * 입력 필드: 이름 / 이메일
     * \[등록하기] 버튼
   * 등록 성공 시 메시지:
     **“🎉 웨이팅 리스트 등록 완료! 출시되면 가장 먼저 알려드릴게요.”**

---

9. **푸터**

   * 링크: 이용약관 / 개인정보 / 블로그 / 커뮤니티 / 상태 페이지
   * SNS 아이콘(트위터, 텔레그램, 디스코드)

---

## 2. 주요 전환 플로우

1. **CTA 버튼 클릭 (Hero, Pricing, Footer 반복 배치)**
2. **폼 입력 (이름 + 이메일)**

   * 연결: 노션 API → 웨이팅 리스트 자동 등록
3. **등록 완료 메시지 출력**

---

## 3. 사용 시나리오 예시 (UI 카드형 표현)

* **개인 투자자:** “퇴근 후 차트 보느라 지친 민수, AI 봇 구독으로 매일 밤 편안한 수면 확보!”
* **트레이더/크리에이터:** “직접 만든 전략을 올려두고, 구독 수익으로 두 번째 월급을 받는 혜진.”
* **개발자:** “API 연동 몇 번 클릭으로 데이터 기반 퀀트 전략을 실험하는 준호.”
* **커뮤니티 운영자:** “팔로워들에게 프리미엄 템플릿을 제공해 부가 수익을 창출하는 리더.”

---

## 4. 인터랙션 요소

* CTA 버튼: Hover 시 포인트 블루(#2F81F7) → 민트(#2CE3B0) 그라데이션
* 카드 Hover 시 뒷면 플립 → 핵심 지표 표시
* FAQ 아코디언: 열림/닫힘 애니메이션
* 다크/라이트 모드 토글 (기본 다크)

---

## 5. 비주얼 가이드

* 색상:

  * 다크 베이스 #0B0F14
  * 포인트 블루 #2F81F7 / 민트 #2CE3B0 / 보라 (김프 전용)
* 폰트: Pretendard(본문), Inter(숫자)
* 아이콘: 심플 라인 아이콘 (보안/지표/랭킹 등)
* 레이아웃: 모바일 우선 카드형, PC에서는 3단/4단 확장

---

👉 위 설계서는 **페이크 도어 테스트용 랜딩 페이지**이므로, 모든 데모/차트/랭킹 데이터는 **예시 UI**로만 제공되고 실제 기능은 동작하지 않음.

---
```
